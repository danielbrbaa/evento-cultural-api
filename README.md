# API ChatBot

Esta API REST foi desenvolvida para gerenciar perguntas e respostas, permitindo o cadastro de perguntas, respostas, locais de busca e o cadastro de usuário.

## Tecnologias Utilizadas

- **Java 17**
- **Spring Boot 3**
- **Spring Data JPA**
- **PostgreSQL**
- **Railway (para deploy)**
