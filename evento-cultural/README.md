# API Evento Cultural

Esta API REST foi desenvolvida para gerenciar eventos culturais, permitindo o cadastro de eventos, artistas, locais e a venda de ingressos.

## Tecnologias Utilizadas

- **Java 17**
- **Spring Boot 3**
- **Spring Data JPA**
- **PostgreSQL**
- **Railway (para deploy)**
